# Local Example App

## Example path

```text
example/test-3dviewer-sdk
```

## Run the example

```bash
cd example/test-3dviewer-sdk
npm install
npm start
```

Then open:

```text
http://localhost:5173
```

## What to test

1. Direct viewer mode with an existing `url`.
2. File pipeline mode with `upload`, `convert`, and `open`.
3. Camera controls, interaction modes, draw modes, toolbar panels, sheets, and cutting planes.
4. 3D markup actions, save/cancel flows, and markup list retrieval.
